# Clinic Pro

📌Descriere

Clinic Pro este o aplicație web dezvoltată în PHP și MySQL pentru managementul activităților dintr-o clinică medicală.

Aplicația permite medicilor să gestioneze pacienți, consultații și programări, oferind o interfață simplă și eficientă pentru utilizarea zilnică.

🚀 Funcționalități
✅ Autentificare utilizatori
✅ Dashboard cu statistici zilnice
✅ Listă pacienți
✅ Adăugare pacient
✅ Căutare pacienț
✅ Vizualizare dosar pacient
✅ Listă consultații
✅ Management programări


🛠 Tehnologii folosite

PHP (backend logic)
MySQL (bază de date)
CSS
